Arrow function

part 2

Pour les arrow functions (fonctions fléchées), on n'écrit plus le mot-clé function. On met juste une flèche (=>) à droite de la liste des arguments.

Exemple:

    const teamES5 = {
        members: ["Vnk", "Dgen"],
        teamName: "Hellloooo",
        teamSummary: function() {
          return this.members.map(function(member) {
            // Uncomment to see the error
            // return `${member} is on team ${this.teamName}`
          });
        }
    };
