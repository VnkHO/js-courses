Arrow function

part 1

Pour les arrow functions (fonctions fléchées), on n'écrit plus le mot-clé function. On met juste une flèche (=>) à droite de la liste des arguments.

Exemple:

const addES5 = function(a, b) { return a + b }

const addES6 = (a, b) => { return a + b }

On peut abréger cette ligne de code encore plus que ce que nous avons actuellement grâce à la fonction fléchée.

const addES6 = (a, b) => a + b

Ainsi, chaque fois qu'on fait face à une seule expression JavaScript, on peut choisir de supprimer les accolades, {}, et également le mot-clé return. On obtient ce qu'on appelle un retour implicite de la fonction.

const double = number => 2 * number;

Lorsqu'on a un seul argument, on peut aussi omettre les parenthèses qui l'entoure.

const twoES6 = () => { return 2 }

const twoES6 = () => 2

Si on n'a pas d'arguments, on doit quand même mettre les parenthèses.
